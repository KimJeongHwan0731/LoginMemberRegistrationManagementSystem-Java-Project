# SignUP/IN Management System Program

## 회원가입 및 로그인 프로그램 소개
- 사용자의 정보를 입력하여 회원가입 완료 후, DataBase와 연동하여 사용자 로그인 flow를 관리하는 프로젝트
1. 회원가입
2. 로그인(로그인 완료 시, 하기 메뉴 선택 가능)

   - 회원정보 수정 
 
   - 회원탈퇴
 
   - 로그아웃
 
3. 회원정보 찾기

   - 아이디 찾기 
 
   - 비밀번호 찾기      
4. 종료
